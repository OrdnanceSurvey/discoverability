GC Data Catalogue 

CSV Header definition 

This document describes the headings used in the Data Catalogue CSV files first published by each of the Geo6 organisations* in January 2019.  This information is also available as a JSON schema HERE.

 

Column _Name 

Obligation 

Description 

Organisation 

Mandatory 

Organisation name or Organisation acronym. 

Dataset_Title 

Mandatory 

Publishing Organisations official title for the dataset. 

Abstract 

Mandatory 

Publishing Organisations description of the dataset. 

INSPIRE_Theme 

Mandatory 

Semi-colon separated list of INSPIRE Theme(s) best describing the dataset. 

INSPIRE_Feature_Type 

Conditional1 

Semi-colon separated list of the INSPIRE Feature Type(s) best describing the dataset.

GC_Theme 

Conditional2 

Semi-colon separated list of keyword values drawn from other Industry vocabularies  e.g. GEMET or industry terms in common usage, where INSPIRE definition is not applicable. 

Constraints 

Optional 

This field outlines any licencing or intellectual property constraints (where known) e.g. if a dataset is published as part of a product or where another organisation has contributed content to the dataset. 

Metadata_published_on_dguk 

Mandatory 

A metadata record for this dataset exists on DGUK.  

 

Data_published 

Mandatory 

A free text field to capture where the data is published.   

This could be a list of products, with a semi-colon separator where there are multiples  

Or  

Where possible a URL to the data will be provided. 

Conditions 

1 When an INSPIRE Theme is either null or set as 'No Applicable Theme'  INSPIRE_Feature_Type must be null 

2 When an INSPIRE Theme is either null or set as 'No Applicable Theme' a Geo6 Theme must not be null. 

 

* British Geological Survey (BGS), Coal Authority (CA), HM Land Registry (LR), UK Hydrographic Office (UKHO), Valuation Office Agency (VoA) and the Ordnance Survey (OS). Collectively known as the Geo6 and working on behalf of the Geospatial Commission.
